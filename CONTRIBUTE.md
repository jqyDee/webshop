### How to contribute to the project
1. Create an Issue with one of the labels (planning, Backend, Frontend) (can be unassigned)
2. Start a new merge request on the `unreviewed` branch (can be done from issue) and assign yourself
3. Move the issue to `In progress`
4. Make your changes
5. For bigger changes assign someone to review your code
6. Move issue to `Testing`
   1. Test Issue (Write Tests+Feedback to Issue)
   2. Move Issue to `TestingFeedback`
   3. Start from 3.
4. If there is no more feedback on the issue it can be merged to `unreviewed`

### Where to ask questions?
If you have any questions related to an issue. Ask directly within the **issue**.
If your question is related to code which is related to an issue ask your
question in the **merge request**

### Commits
Try to follow these [commit conventions](https://www.conventionalcommits.org/en/v1.0.0).
